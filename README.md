# Airport Finder by Clawz

A private aviation-style Progressive Web App for fast airport code lookup.

## Release 2.0.0 Sprint 4/5

Features:

- OpenFlights-based airport database
- Search by IATA, ICAO, city, airport name and country
- Exact IATA matches are ranked first
- Exact ICAO matches are ranked second
- Airport detail card
- Apple Maps and Google Maps links
- Copy IATA button
- Timezone and local time display
- PWA service worker
- Clawz logo integration

Airport records included: 7,697

Data source: OpenFlights airport database.
